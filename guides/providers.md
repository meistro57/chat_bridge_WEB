# AI Providers

Chat Bridge supports multiple AI providers:

- **OpenAI**: Requires an API key from platform.openai.com. Supports GPT-4o, GPT-4o-mini, etc.
- **Anthropic**: Requires an API key from console.anthropic.com. Supports Claude 3.5 Sonnet, etc.
- **Gemini**: Requires a Google AI Studio key.
- **Ollama**: Connects to a local Ollama instance (no key required).
- **LM Studio**: Connects to a local LM Studio server.
