# Getting Started

Welcome to Chat Bridge Web! 

1. **Set up API Keys**: Click the "Keys" button in the header to enter your API keys.
2. **Select Personas**: Click on Agent A and Agent B to choose their personalities from the library.
3. **Choose Providers**: Select which AI model should power each agent.
4. **Start Conversation**: Enter a starter message and click "Launch Conversation".

The agents will then converse in real-time via WebSockets.
